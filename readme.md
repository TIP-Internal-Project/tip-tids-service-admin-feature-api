# Admin Feature
  Engagement API for Admin Management

## Repository
```
git clone https://github.com/esonpaguia/admin-feature.git
```

## Tech Stack
- node
- docker
- docker-compose

## Build & Deploy
- Startup
```
docker-compose up -d --build
```
- Open browser and go to http://localhost:3000
- Shutdown
```
docker-compose down -v --rmi all
```
